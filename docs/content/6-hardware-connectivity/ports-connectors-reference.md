#  Hardware Interface Specifications

![USBridge Physical Ports Reference](https://raw.githubusercontent.com/USBridge-Technologies/USBridge-KVM-2.0/main/docs/assets/Ports.png)


| Port | Type | What is it for? / How to use it |
| :--- | :---: | :--- |
| **USB-C (OTG / Power)** | Input / Output | Powers the KVM device (5V/3A) and emulates a physical keyboard, mouse, and virtual drive arrays. |
| **USB-C (Host)** | Input | Dedicated port for connecting the external HDMI video capture dongle. |
| **8-pin GPIO Port** | Control | Connects to the external Power Management Module. |
| **MicroSD Slot** | Storage | SD card slot. |
| **Micro HDMI Port** | Output | Used to connect a local monitor. |
