# Core Hardware Architecture

The USBridge appliance is engineered as a modular hardware platform, balancing the high compute performance required for real-time video processing with a highly compact physical form factor. The architecture operates as a dedicated out-of-band management node, ensuring absolute isolation between the host server and the KVM system logic.

---

## 1. Core Processing Module

The primary compute engine driving the appliance is based on the Radxa Zero 3 single-board computer architecture:

| System Component | Technical Specification & Implementation |
| :--- | :--- |
| **System-on-Chip (SoC)** | Rockchip RK3566 (Quad-Core ARM Cortex-A55)[cite: 1] |
| **Base Board Module** | Radxa Zero 3W variant[cite: 1] |
| **System Memory (RAM)** | 1–2 GB LPDDR4X (Capacity subject to specific SKU revision; partially allocated for high-speed Virtual Media caching)[cite: 1] |
| **Internal Flash Storage** | High-speed on-board eMMC (Dedicated to the secure system image, system logs, and snapshot database persistence)[cite: 1] |

---

## 2. Thermal Management Architecture

To guarantee long-term mechanical reliability and stable operation within dense server racks or high-density cabinets, the platform employs a specialized hybrid cooling design[cite: 1]:

* **Thermal Energy Dissipation:** Heat generated by the Rockchip RK3566 SoC and high-frequency LPDDR4X memory modules is absorbed through a custom-milled internal aluminum heatsink[cite: 1].
* **Active Protection Fan:** To prevent thermal throttling during continuous maximum workloads—such as high-frame-rate Moonlight video encoding or intensive Copy-on-Write (CoW) block storage caching—an active cooling fan is integrated into the chassis[cite: 1].
* **Environmental Boundaries:** This thermal engineering maintains predictable performance in ambient temperatures ranging from 0°C to 50°C, preserving metadata integrity during prolonged deployment scripts.

---

## 3. Local Telemetry & Control Block

For immediate, on-site diagnostics without network dependency or remote client app access, the hardware integrates a dedicated local interface block:

* **Integrated Color IPS LCD Panel:** A built-in 240x240 resolution screen surfaces critical telemetry right at the rack, including network IP assignments, current interface states, and hardware health metrics[cite: 1].
* **Physical Rotary Encoder Navigation:** A physical rotary encoder with push-button confirmation allows operators to execute direct actions—such as navigating menus, switching tracking modes, or triggering hardware-level snapshots—entirely independent of the client software layer[cite: 1].
